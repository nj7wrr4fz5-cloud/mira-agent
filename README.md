# 🤖 Mira Agent

AI-ассистент для твоего ПК с подключением к Mira AI.

## Возможности

- **AI Chat** — Общение с ИИ-ассистентом
- **Планировщик** — Автоматическая отправка сообщений по расписанию (cron)
- **Группы** — Управление Telegram-группами
- **Боты** — Настройка и управление ботами
- **Статистика** — Метрики и аналитика
- **Автообновление** — Автоматические обновления через GitHub Releases
- **Соединение с Mira** — Ты всегда онлайн!

## Установка

```bash
# Клонировать репозиторий
git clone https://github.com/nj7wrr4fz5-cloud/mira-agent.git
cd mira-agent

# Установить зависимости
npm install

# Запустить в режиме разработки
npm run dev

# Собрать приложение
npm run build

# Создать .exe (Windows)
npm run package
```

## Структура проекта

## Использование

1. Запусти приложение
2. Открой вкладку **AI Chat** для общения с Мирой
3. Настрой расписание в **Планировщик**
4. Добавь группы в **Группы**
5. Настрой ботов в **Боты**
6. Проверь статистику в **Статистика**
7. Настрой автообновление в **Настройки**

## Обновления

Приложение автоматически проверяет обновления при запуске и уведомляет о доступных версиях через GitHub Releases.
